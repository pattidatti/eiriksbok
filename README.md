# Gravity Lærebok 📚

Dette er kildekoden til den digitale læreboken.

## 🌍 Nettside
Du kan se den ferdige nettsiden her: **[https://pattidatti.github.io/Eiriksbok/](https://pattidatti.github.io/Eiriksbok/)**

## 🛠️ Utvikling

1.  Installer avhengigheter:
    ```bash
    npm install
    ```
2.  Start utviklingsserver:
    ```bash
    npm run dev
    ```
