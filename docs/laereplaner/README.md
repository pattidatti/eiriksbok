## Om kjennetegn på måloppnåelse

Kjennetegnene er veiledende kvalitetsbeskrivelser fra Udir, formulert for tre nivåer:

- **Karakter 2** — Lav kompetanse
- **Karakter 4** — God kompetanse
- **Karakter 6** — Framifrå kompetanse

Kjennetegnene må brukes sammen med kompetansemålene i læreplanen. De er ment som støtte for standpunktvurdering.
